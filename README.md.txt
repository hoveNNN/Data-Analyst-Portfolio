#  Sales Analysis Project

## Project Overview
This analysis explores sales data from a global superstore, identifying trends in:
- Regional performance
- Product category sales
- Monthly revenue patterns

## Tools Used
| Tool          | Purpose                      |
|---------------|------------------------------|
| Excel         | Data cleaning                |
| SQL (SQLite)  | Data analysis and queries    |
| Power BI      | Interactive visualization    |

## Key Insights
1. **Technology products** generated the highest sales ($500K)
2. **December** was the peak sales month (holiday season)
3. The **West region** had 30% higher profit margins than others
